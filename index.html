1. 個人簡介和申請動機

WeHelp 的各位老師好！我叫郭品妤，目前在資訊服務出版業做資料分析，2020/03~2023/02年做的是零件業的西班牙語開發業務， 當時工作中接觸到 Excel 分析預測圖表、PowerBI 後，發覺自己對資料很有興趣，便全職參加了五個多月的勞動部產業新尖兵計畫。

轉職班教授的內容涵蓋 Linux、Python、SQL、視覺化、分佈式系統、Git、GCP，結束同學們走的路很多元，我開始探索自己的方向， 便在 Udemy, Hahow 購買一些課程學習，因為這些課程讓我接觸到了 Machine learning、Deep Learning，身邊人覺得無聊的資料，我卻覺得很有趣， 也做了幾個 Kaggle 作品。

當時心裡想著除了轉職班的團體作品外，我要另外部署一個自己的作品網站去面試，於是使用 Streamlit，爬蟲抓取財報上傳 Github、下載讀取、Plotly 繪圖， 網站部署在 Streamlit 的平臺，手動每一兩季更新一次。

2024 過年後進到資訊服務出版業，工作主要使用 Pandas 和 Numpy 進行資料清整、依照客戶產業，下 SQL 條件給資料工程師、爬蟲抓產業資料、與同事共同維護機器學習模型，資料文字向量化預測並優化、 Tableau、Excel、PPT 報表建立。公司的資料工程師使用的是 Jieba+lstm 預測文字型資料，我和同期同事也試著用 Openai Embedding+Llama，跟他們一起工作我學習到很多，也更確定自己想往這個方向。

自己是文組背景，我會在開放式平臺複習和學習基礎學科，因為大學通識學過的統計課很基礎，我學習了統計學-李柏堅(CUST OCW)、統計學-唐麗英(NYCU OCW)，正在進行 線性代數-蘇柏青(NTU OCW)、微積分-張旭(Youtube 張旭無限教室)，希望明年有機會跟大家一起學習。
